# Almacenamiento

Antes de instalar el launcher, es importante conocer cuánto espacio necesitarás

| Estado                                          | Tamaño  |
| ----------------------------------------------- | ------- |
| Instalador descargado                           | 905 MB  |
| Launcher instalado                              | 1.43 GB |
| Launcher actualizado (mods, recursos y shaders) | 1.78 GB |

> [!NOTE]
> Después de instalar el launcher puedes eliminar el instalador para liberar espacio.

---

# Crear perfil y subir skin

Envía tu skin al chat con **Chami**.

Lo ideal es usar la misma skin que utilizarás dentro del servidor.

Después recibirás:

* Skin registrada
* Imagen de perfil
* Comando para aplicarla

## Ejemplo

✅ Skin registrada correctamente

| Skin                                                                  | Perfil                                                                   |
| --------------------------------------------------------------------- | ------------------------------------------------------------------------ |
| <img src="https://launcher-chami.web.app/Skins/Chami.png" width="64"> | <img src="https://launcher-chami.web.app/Perfiles/Chami.png" width="64"> |

### Modelo Classic

```mcfunction
/skin set web classic "https://launcher-chami.web.app/Skins/Chami.png"
```

### Modelo Slim

```mcfunction
/skin set web slim "https://launcher-chami.web.app/Skins/Chami.png"
```

También podrás visualizar tu perfil directamente desde el launcher.

<img width="137" src="https://github.com/user-attachments/assets/4d84c478-b535-46cb-a50d-ef8d443955e7" />

# Actualizar el launcher

Ahora falta descargar:

* Mods
* Recursos
* Shaders
* Archivos adicionales

En la esquina superior izquierda encontrarás el botón **Actualizar**.

<img width="1366" src="https://github.com/user-attachments/assets/12762681-4550-437a-bd4d-7205553b6908" />

Al presionarlo aparecerá algo como esto:

<img width="601" src="https://github.com/user-attachments/assets/a795c5e1-7397-4676-9e42-dab274dbe72d" />

> [!NOTE]
> Esto puede tardar algunos minutos ya que el launcher verifica todos los archivos necesarios.

Una vez finalizado, podrás jugar normalmente.
